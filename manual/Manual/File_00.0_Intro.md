BarBuddy V5 - The Complete Home Bartender's Manual
"Drink Better, Smoke Often."
Welcome to V5. This is not just a recipe book. It is a comprehensive operating system for your home bar. It covers the science of flavor, the art of presentation, and the technique of smoking cocktails.
HOW TO USE THIS MANUAL
This manual is modular. You don't need to read it cover-to-cover.
Start with File 01.0 & 02.0: If you are new to making drinks or smoking them.
Jump to File 03.x: If you just want to make a drink right now.
Consult File 15.0: If your drink tastes bad and you need to fix it.
Check File 20.0: For quick measurements during a party.
