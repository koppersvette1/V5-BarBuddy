SMOKE COMPATIBILITY LEGEND
Symbol
Meaning
Notes
✅ Yes
Excellent for smoking
Use recommended wood and timing
⚠️ Light Only
Can smoke but be very careful
Use lighter woods, shorter times
❌ No
Do not smoke
Refreshing/delicate flavors conflict with smoke
❌ NEVER
Never smoke
Contains mint or similar - smoke ruins the drink
✅ REQUIRED
Smoking is essential
These are specialty smoked cocktails

SMOKING GUIDE BY WOOD TYPE
Cherry Wood (Sweet Smoke)
Best For: Old Fashioned, Whiskey Sour, Paper Plane, Amaretto Sour, Smoked Old Fashioned
Time: 8-10 seconds
Oak Wood (Barrel Smoke)
Best For: Manhattan, Vieux Carré, Sidecar, Dark 'n' Stormy, Smoked Manhattan
Time: 8-10 seconds
Grapevine Wood (Fruity Smoke)
Best For: Negroni, Smoked Boulevardier
Time: 8-10 seconds
Apple Wood (Light Fruity Smoke)
Best For: Martini, Daiquiri, Bee's Knees, Hemingway Daiquiri, Pisco Sour, Mai Tai, Jungle Bird, Vesper, Last Word
Time: 10-12 seconds (light smoke)
Mesquite Wood (Strong Savory Smoke)
Best For: Margarita, Paloma, Oaxaca Old Fashioned
Time: 3-7 seconds MAX (very strong)
Pear Wood (Delicate Smoke)
Best For: Penicillin, Smoked Penicillin
Time: 10-12 seconds
Maple/Pecan Wood (Sweet/Nutty Smoke)
Best For: Espresso Martini, Gold Rush
Time: 8-10 seconds
