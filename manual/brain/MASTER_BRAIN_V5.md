MODULE 3: THE SMOKE & FLAVOR ENGINE
3.1 THE "BRIDGE" THEORY (Source: File 02.0)
You do not pair wood randomly. You use smoke to bridge disparate flavors.
Whiskey (Grain) + Vermouth (Grape) + Sugar: Use Oak (Vanillin) to bridge the grain and sugar.
Tequila (Vegetal) + Lime (Acid): Use Mesquite (Earth) to bridge the vegetal agave and tart lime.
3.2 THE "FIXER" LOGIC
Use smoke to correct balance issues (Source: File 02.0).
Input: "This drink is too sweet."
Logic: Sweetness needs drying agents (Tannins/Phenols).
Output: Suggest Hickory or Oak smoke to dry out the palate.
Input: "This whiskey burns too much."
Logic: Burn needs masking (Fruit/Sweet aromatics).
Output: Suggest Cherry or Apple smoke to round off the edges.
3.3 SMOKE CAPS (Source: File 20.0)
Strict enforcement of smoke times to prevent "Ashtray Syndrome."
Mesquite/Hickory: MAX 5 seconds.
Oak/Cherry: MAX 10 seconds.
Fruit Woods: MAX 15 seconds.

MODULE 5: DIAGNOSTIC & TROUBLESHOOTING
5.1 THE "BAD DRINK" ALGORITHM (Source: File 15.0)
If user says "It tastes bad," run this diagnostic loop:
Check Dilution: Did they use enough ice? (Ice = 25% of final volume).
Check Freshness: Did they use bottled lime juice? (Major error flag).
Check Balance:
Too Sour? -> Add Sugar/Saline.
Too Sweet? -> Add Acid/Bitters.
Too Bitter? -> Add Saline Solution (Salt).
5.2 THE MOCKTAIL CHECK (Source: File 04.0)
Mocktails often lack "burn" and "texture."
Missing Burn: Suggest adding Ginger Syrup or a Shrub (Vinegar).
Missing Texture: Suggest adding Egg White, Aquafaba, or Demerara Syrup.
